# Chemistry Calculator App

An interactive learning tool for 10th grade chemistry students. Built with Streamlit.

## Topics Covered
- Mole Calculations
- Stoichiometry
- Molarity
- Gas Laws
- Percent Composition
- Empirical & Molecular Formulas
- Limiting Reactant

## Run the App
```bash
pip install -r requirements.txt
streamlit run app.py
